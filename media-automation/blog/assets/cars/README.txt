[ 차량 이미지 폴더 ]

여기에 현대·제네시스 공식 홈페이지에서 받은 차량 이미지를 넣어주세요.
(사장님이 사용 권한을 가진 공식 이미지)

■ 넣는 방법
1) blog/config/car-images.json 의 "url" 주소로 들어갑니다.
2) 해당 모델 이미지를 내려받습니다.
3) 같은 json 의 "file" 이름 그대로 이 폴더에 저장합니다.
   예) 그랜저 → grandeur.jpg  /  싼타페 → santafe.jpg

■ 파일이 없을 때
- 원고에는 "현대 OOO 공식 이미지 자리" 안내 카드가 대신 들어갑니다.
  (어디에 어떤 사진을 넣을지 한눈에 보이도록 표시됨)
- 나중에 파일만 넣고 다시 생성하면 실제 사진으로 자동 교체됩니다.

지원 확장자: .jpg .jpeg .png .webp
